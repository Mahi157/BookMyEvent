# BookMyEvent вЂ“ College Event Discovery & Booking Platform

## Overview

BookMyEvent is a responsive web-based platform designed to help students discover, explore, and register for various college events from a single portal. The platform provides a user-friendly interface for browsing Tech Fests, Cultural Events, Workshops, Music Concerts, Sports Tournament, Hackathons, and other campus activities.

The project aims to simplify event discovery and registration while offering an engaging user experience through modern UI components such as carousels, category-wise event pages, search functionality, and registration forms.
---

## Features

### Event Discovery

* Browse events across multiple categories
* View event details including date, venue, and registration options
* Dedicated pages for each event category

### Categories Supported

* Tech Events
* Cultural Events
* Workshops
* Music Events
* Sports Events
* Recommended Events
* Upcoming Events

### Search Functionality

* Search events using keywords
* Quick navigation to relevant events

### Event Registration

* Registration form for event booking
* User-friendly booking workflow
* Registration confirmation interface

### Interactive User Interface

* Hero image carousel
* Category-wise event cards
* Smooth horizontal scrolling
* Hover effects and transitions
* Responsive design

### Additional Features

* Location Selection Popup
* Sign-In Interface
* Newsletter Subscription Section
* Footer with Quick Navigation Links
